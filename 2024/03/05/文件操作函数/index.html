文件操作相关函数详解

c语言文件按编码方式分为二进制文件和ASCII文件。 c语言用文件指针标识文件，对文件进行操作时，需要遵从打开(创建)->读写->关闭的流程。

fopen函数

在C语言中，fopen 函数用于打开一个文件。这个函数需要两个参数：文件名和打开模式。如果打开成功，fopen 函数返回一个 FILE * 指针，这个指针可以用于后续的文件操作；如果打开失败，它返回 NULL。其函数原型如下所示。

FILE *fopen(char *filename, char *mode);

mode即文件打开的方式，由两类字符构成：一类是打开文件的类型，包括文本文件(text)和二进制文件(binary)，如果不指定文件的类型，则默认为文本文件；另一类是操作类型，read、write、append。

以下是一些常见的打开模式：

  • "r"：以读取模式打开文件。文件必须存在。
  • "w"：以写入模式打开文件。如果文件不存在，创建新文件；如果文件存在，删除文件内容，进行覆盖。
  • "a"：以追加模式打开文件。如果文件不存在，创建新文件；如果文件存在，将数据追加到文件末尾。
  • "r+"：以读取/写入模式打开文件。文件必须存在。
  • "w+"：以读取/写入模式打开文件。如果文件不存在，创建新文件；如果文件存在，删除文件内容，进行覆盖。
  • "a+"：以读取/追加模式打开文件。如果文件不存在，创建新文件；如果文件存在，将数据追加到文件末尾。
  • "wb+"：以读取/写入模式打开二进制文件。如果文件不存在，创建新文件；如果文件存在，删除文件内容，进行覆盖。

fclose函数

在C语言中，fclose 函数用于关闭一个打开的文件。这个函数的参数是一个 FILE * 指针，这个指针必须是由 fopen、freopen 或 tmpfile 函数返回的。如果关闭成功，fclose 函数返回 0；如果发生错误，它返回 EOF。

关闭文件是很重要的，因为它可以释放系统资源，确保所有的输出都被写入到文件中，并且可以防止文件被意外修改或删除。

字符读写函数

fgetc函数

在C语言中，fgetc 函数用于从一个打开的文件中读取一个字符。这个函数的参数是一个 FILE * 指针，这个指针必须是由 fopen、freopen 或 tmpfile 函数返回的。如果读取成功，fgetc 函数返回读取的字符，同时将读写位置指针指向下一个字符；如果读取失败或到达文件末尾，它返回 EOF。

fputc函数

在C语言中，fputc 函数用于向一个打开的文件中写入一个字符。这个函数需要两个参数：要写入的字符和一个 FILE * 指针，这个指针必须是由 fopen、freopen 或 tmpfile 函数返回的。如果写入成功，fputc 函数返回写入的字符；如果发生错误，它返回 EOF。其函数原型为int fputc(char ch,FILE *fp)

字符串读写函数

fgets函数

在C语言中，fgets函数用于从指定的流中读取一行文本，直到遇到换行符、EOF（文件结束标志）或者读取了指定数量的字符，读取成功后返回指向字符串的指针，读到文件末尾或者出错，则返回NULL。
注意： fgets函数最多读取n-1个字符长度的字符串，读取完后会在字符串的末尾加上串结束标志'\0'。

fputs函数

数据块读写函数

fread函数

在C语言中，fread 函数用于从一个打开的文件中读取数据。这个函数需要四个参数：一个指向存储数据的缓冲区的指针，元素的大小，元素的数量，以及一个 FILE * 指针，这个指针必须是由 fopen、freopen 或 tmpfile 函数返回的。如果读取成功，fread 函数返回实际读取的元素数量，注意不是字节个数；如果读取失败或到达文件末尾，它返回的数量可能小于请求的数量。其函数原型如下：
unsigned fread(void *ptr,unsigned size,unsigned n,FILE *fp);

格式化读写函数

fscanf函数

在C语言中，fscanf 函数用于从一个打开的文件中读取格式化的输入。这个函数需要至少两个参数：一个 FILE * 指针，这个指针必须是由 fopen、freopen 或 tmpfile 函数返回的，以及一个格式字符串，这个字符串描述了要读取的数据的格式。fscanf 函数还可以接受额外的参数，这些参数是指向变量的指针，这些变量将存储读取的数据。

如果读取成功，fscanf 函数返回实际读取的数据项的数量；如果到达文件末尾或发生错误，它返回 EOF。

fprintf函数

在C语言中，fprintf 函数用于向一个打开的文件中写入格式化的输出。这个函数需要至少两个参数：一个 FILE * 指针，这个指针必须是由 fopen、freopen 或 tmpfile 函数返回的，以及一个格式字符串，这个字符串描述了要写入的数据的格式。fprintf 函数还可以接受额外的参数，这些参数是要写入的数据。
如果写入成功，fprintf 函数返回写入的字符数量。如果发生错误，返回EOF。

注意：fprintf函数总是以字符串的形式将数据信息存放到文件中，而不是以数值的形式，不管打开的是文本文件还是二进制文件。

feof函数

在C语言中，feof 函数用于检查一个打开的文件是否到达了文件末尾。这个函数的参数是一个 FILE * 指针，这个指针必须是由 fopen、freopen 或 tmpfile 函数返回的。如果到达了文件末尾，feof 函数返回非零值；如果没有到达文件末尾，它返回零。
其函数原型为：int feof(FILE *fp);

文件定位读写

文件的读写位置指针最小值为0，最大值为文件的长度。文件的位置指针位于文件首部，随着数据的读出和写入，文件的位置指针会发生移动。

rewind函数

在C语言中，rewind 函数用于将一个打开的文件的位置指针重置到文件的开始。这个函数需要一个 FILE * 指针作为参数，这个指针必须是由 fopen、freopen 或 tmpfile 函数返回的。其函数原型如下：void rewind(FILE *fp);

rewind 函数没有返回值。如果你需要检查操作是否成功，可以使用 ferror 或 feof 函数。

fseek函数

在c语言中，fseek函数用于将一个打开的文件的读写位置指针移动到特定的位置。
其函数原型为：int fseek(FILE *fp, long offset, int whence)，即将位置指针移动到距离whence的offset字节处。offset取值为正，即表示将文件位置指针移动到whence后面；为负，则表示将指针移动到whence的前面。

注意： fseek函数一般用于二进制文件。在文本文件中由于要进行转换，故往往计算的位置会出现错误。

whence取值如下：

whence的常量值 对应数值 含义
SEEK_SET 0 文件开始处
SEEK_CUR 1 文件位置指针的当前位置
SEEK_END 2 文件的末尾

若操作成功，则fseek函数返回0；否则返回非0值。
